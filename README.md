# MCP-AI-Automation-Project

An AI-powered automation system built using MCP Servers, Cursor IDE, n8n, Pipedream, OpenAI, LinkedIn APIs, Gmail APIs, and ElevenLabs Voice AI.

This project demonstrates real-world AI integrations capable of:
- Automating LinkedIn posts
- Sending Gmail applications
- Generating AI summaries
- Creating X (Twitter) posts
- Voice AI calling agents
- AI-powered learning path generation
- Railway/train assistant workflows
