# MCP-AI-Automation-Project

An AI-powered automation system built using MCP Servers, Cursor IDE, n8n, Pipedream, OpenAI, LinkedIn APIs, Gmail APIs, and ElevenLabs Voice AI.

This project demonstrates real-world AI integrations capable of:
- Automating LinkedIn posts
- Sending Gmail applications
- Generating AI summaries
- Creating X (Twitter) posts
- Voice AI calling agents
- AI-powered learning path generation
- Railway/train assistant workflows


## Features

- MCP Server Integrations
- LinkedIn Automation
- Gmail Automation
- AI Content Generation
- Voice AI Agent using ElevenLabs
- n8n Workflow Automation
- OpenAI Prompt Engineering
- AI-based Learning Path Generator
- Railway Assistant using MCP Tools

## Technologies & Tools Used

- Cursor IDE
- MCP Servers
- OpenAI APIs
- n8n
- Pipedream
- LinkedIn MCP Tools
- Gmail MCP Tools
- ElevenLabs
- Google Sheets
- X (Twitter) API

## Project Workflow

1. Integrated MCP tools inside Cursor IDE
2. Connected LinkedIn, Gmail, Railway, and ElevenLabs tools
3. Built AI agents using prompts and OpenAI models
4. Created n8n workflows for automation
5. Generated automated LinkedIn and X posts
6. Created voice AI calling agents
7. Automated learning path generation system
